Python Klassen

Einführung in Klassen

Klassen sind ein grundlegendes Konzept in der objektorientierten Programmierung. Sie ermöglichen es, benutzerdefinierte Datentypen zu erstellen und Methoden und Attribute zu definieren. Weitere Informationen zu Klassen in Python finden Sie in der offiziellen Python-Dokumentation.

Definition und Instanziierung von Klassen

Eine Klasse wird mit dem Schlüsselwort class definiert. Methoden und Attribute werden innerhalb des Klassenblocks definiert.

# Definition einer einfachen Klasse
class Person:
    def __init__(self, name, alter):
        self.name = name
        self.alter = alter

    def begruessung(self):
        return f"Hallo, mein Name ist {self.name} und ich bin {self.alter} Jahre alt."

# Instanziierung der Klasse
person1 = Person("Alice", 30)
print(person1.begruessung())

Weitere Instanzen erstellen

Man kann mehrere Instanzen einer Klasse erstellen, um verschiedene Objekte zu repräsentieren.

# Erstellen einer weiteren Instanz
person2 = Person("Bob", 25)
print(person2.begruessung())

Klassenattribute und -methoden

Klassen können Attribute und Methoden enthalten, die für alle Instanzen der Klasse gleich sind.

# Definition von Klassenattributen und -methoden
class Auto:
    anzahl_raeder = 4  # Klassenattribut

    def __init__(self, marke, modell):
        self.marke = marke
        self.modell = modell

    def beschreibung(self):
        return f"Das Auto ist ein {self.marke} {self.modell} mit {Auto.anzahl_raeder} Rädern."

# Instanziierung der Klasse
auto1 = Auto("Toyota", "Corolla")
print(auto1.beschreibung())

Vererbung

Vererbung ermöglicht es, eine neue Klasse auf der Basis einer bestehenden Klasse zu erstellen. Die neue Klasse erbt die Attribute und Methoden der bestehenden Klasse.

# Definition einer Basisklasse
class Tier:
    def __init__(self, name):
        self.name = name

    def geraeusch(self):
        pass  # Wird in der abgeleiteten Klasse überschrieben

# Definition einer abgeleiteten Klasse
class Hund(Tier):
    def geraeusch(self):
        return "Wuff"

# Instanziierung der abgeleiteten Klasse
hund = Hund("Bello")
print(f"{hund.name} sagt {hund.geraeusch()}")

Magische Methoden

Magische Methoden (oder Dunder-Methoden) sind spezielle Methoden, die bestimmte Funktionen in Python implementieren, z.B. __init__ oder __str__.

# Verwendung magischer Methoden
class Kreis:
    def __init__(self, radius):
        self.radius = radius

    def __str__(self):
        return f"Kreis mit Radius {self.radius}"

    def flaeche(self):
        return 3.14 * (self.radius ** 2)

kreis = Kreis(5)
print(kreis)  # Ausgabe: Kreis mit Radius 5
print(f"Fläche: {kreis.flaeche()}")  # Ausgabe: Fläche: 78.5
