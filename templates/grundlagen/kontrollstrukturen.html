Python Kontrollstrukturen

Einführung in Kontrollstrukturen

Kontrollstrukturen sind essenziell, um den Ablauf eines Programms zu steuern. Sie ermöglichen bedingte Ausführung, Schleifen und vieles mehr. Weitere Informationen zu Kontrollstrukturen in Python finden Sie in der offiziellen Python-Dokumentation.

If-Else-Bedingungen

If-Else-Bedingungen erlauben es, Code abhängig von bestimmten Bedingungen auszuführen.

# If-Else-Bedingung
x = 10
if x > 5:
    print("x ist größer als 5")
else:
    print("x ist kleiner oder gleich 5")

For-Schleifen

For-Schleifen werden verwendet, um über eine Sequenz (z.B. eine Liste, ein Tuple oder ein String) zu iterieren.

# For-Schleife
for i in range(5):
    print(i)

While-Schleifen

While-Schleifen werden verwendet, um Code so lange auszuführen, wie eine bestimmte Bedingung erfüllt ist.

# While-Schleife
count = 0
while count < 5:
    print(count)
    count += 1

Verschachtelte Schleifen

Man kann Schleifen ineinander verschachteln, um komplexe Iterationen zu ermöglichen.

# Verschachtelte Schleifen
for i in range(3):
    for j in range(2):
        print(f"i = {i}, j = {j}")

Break- und Continue-Anweisungen

Mit den Anweisungen break und continue kann man den Ablauf von Schleifen steuern.

# Break-Anweisung
for i in range(5):
    if i == 3:
        break
    print(i)

# Continue-Anweisung
for i in range(5):
    if i == 3:
        continue
    print(i)

Else in Schleifen

Man kann in Python eine else-Klausel zu Schleifen hinzufügen, die ausgeführt wird, wenn die Schleife nicht durch ein break unterbrochen wird.

# Else in Schleifen
for i in range(5):
    print(i)
else:
    print("Schleife beendet")

# Else mit Break
for i in range(5):
    if i == 3:
        break
    print(i)
else:
    print("Schleife beendet")

Pass-Anweisung

Die pass-Anweisung kann verwendet werden, um eine leere Codeblock zu erstellen. Sie wird oft als Platzhalter verwendet.

# Pass-Anweisung
for i in range(5):
    if i == 3:
        pass  # Platzhalter, tut nichts
    print(i)
