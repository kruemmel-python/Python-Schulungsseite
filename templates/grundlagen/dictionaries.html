Python Dictionaries

Einführung in Dictionaries

Dictionaries sind eine grundlegende Datenstruktur in Python, die es ermöglicht, Schlüssel-Wert-Paare zu speichern. Sie sind veränderbar und können verschiedene Datentypen als Schlüssel oder Werte enthalten.

Weitere Informationen zu Dictionaries in Python finden Sie in der offiziellen Python-Dokumentation.

Erstellen von Dictionaries

Dictionaries können erstellt werden, indem man die Schlüssel-Wert-Paare in geschweifte Klammern {} schreibt, getrennt durch Kommas. Jeder Schlüssel wird durch einen Doppelpunkt von seinem Wert getrennt.

# Ein Dictionary mit verschiedenen Datentypen
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
print(person)

Zugriff auf Werte in Dictionaries

Auf Werte in einem Dictionary kann mit Hilfe ihrer Schlüssel zugegriffen werden.

# Zugriff auf Werte über Schlüssel
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
name = person["name"]
print(name)

Hinzufügen und Entfernen von Schlüssel-Wert-Paaren

Man kann Schlüssel-Wert-Paare zu einem Dictionary hinzufügen oder daraus entfernen.

# Hinzufügen eines neuen Schlüssel-Wert-Paares
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
person["beruf"] = "Ingenieur"
print(person)

# Entfernen eines Schlüssel-Wert-Paares
del person["stadt"]
print(person)

Dictionaries durchlaufen

Man kann ein Dictionary mit Hilfe einer Schleife durchlaufen. Dabei kann man auf die Schlüssel, die Werte oder beides zugreifen.

# Durchlaufen eines Dictionaries
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
for schluessel, wert in person.items():
    print(f"Schlüssel: {schluessel}, Wert: {wert}")

Weitere Dictionary-Operationen

Es gibt viele nützliche Methoden und Funktionen, die man mit Dictionaries verwenden kann. Hier sind einige der wichtigsten:

Überprüfen, ob ein Schlüssel vorhanden ist

Mit dem Schlüsselwort in kann man überprüfen, ob ein bestimmter Schlüssel in einem Dictionary vorhanden ist.

# Überprüfen, ob ein Schlüssel vorhanden ist
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
if "name" in person:
    print("Der Schlüssel 'name' ist im Dictionary vorhanden.")

Standardwert für einen Schlüssel abrufen

Mit der Methode get() kann man den Wert für einen Schlüssel abrufen und einen Standardwert zurückgeben, wenn der Schlüssel nicht vorhanden ist.

# Standardwert für einen Schlüssel abrufen
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
beruf = person.get("beruf", "Beruf nicht angegeben")
print(beruf)

Alle Schlüssel und Werte abrufen

Mit den Methoden keys() und values() kann man alle Schlüssel und Werte eines Dictionaries abrufen.

# Alle Schlüssel und Werte abrufen
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
schluessel = person.keys()
werte = person.values()
print("Schlüssel:", list(schluessel))
print("Werte:", list(werte))

Ein Dictionary kopieren

Mit der Methode copy() kann man ein Dictionary kopieren.

# Ein Dictionary kopieren
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
person_kopie = person.copy()
print(person_kopie)

Alle Schlüssel-Wert-Paare entfernen

Mit der Methode clear() kann man alle Schlüssel-Wert-Paare aus einem Dictionary entfernen.

# Alle Schlüssel-Wert-Paare entfernen
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
person.clear()
print(person)

Schlüssel-Wert-Paare zusammenfügen

Mit der Methode update() kann man zwei Dictionaries zusammenfügen.

# Schlüssel-Wert-Paare zusammenfügen
person = {
    "name": "Alice",
    "alter": 30,
    "stadt": "Berlin"
}
zusatz_info = {
    "beruf": "Ingenieur",
    "hobby": "Lesen"
}
person.update(zusatz_info)
print(person)
