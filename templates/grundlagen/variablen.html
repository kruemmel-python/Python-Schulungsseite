Python Variablen

Was sind Python Variablen?

Variablen sind ein grundlegendes Konzept in der Programmierung. Sie ermöglichen es, Daten in einem Programm zu speichern und zu manipulieren. In Python müssen Variablen nicht deklariert werden, bevor sie verwendet werden.

Weitere Informationen zu Variablen in Python finden Sie in der offiziellen Python-Dokumentation.

Variable in Python – eine Typfrage

In Python können Variablen verschiedene Datentypen speichern. Hier sind einige der gängigsten Datentypen:

  • int: Ganze Zahlen, z.B. 5
  • float: Fließkommazahlen, z.B. 3.14
  • str: Zeichenketten, z.B. "Hallo"
  • bool: Boolesche Werte, z.B. True oder False
  • list: Listen, z.B. [1, 2, 3]
  • dict: Wörterbücher, z.B. {"name": "Alice", "alter": 30}

Weitere Informationen zu Datentypen finden Sie in der offiziellen Python-Dokumentation.

Beispiele für Variablen

Hier sind einige Beispiele für die Verwendung von Variablen in Python:

# Eine Variable mit einem Ganzzahlwert
x = 5
print(x)

# Eine Variable mit einem Textwert
name = "Alice"
print(name)

# Eine Variable mit einem Fließkommawert
pi = 3.14159
print(pi)

# Eine Variable mit einer Liste
fruits = ["Apfel", "Banane", "Kirsche"]
print(fruits)

Regeln für die Namensvergabe von Python Variablen

  • Variablennamen müssen mit einem Buchstaben (a-z, A-Z) oder einem Unterstrich (_) beginnen.
  • Der Rest des Namens kann Buchstaben, Ziffern (0-9) oder Unterstriche enthalten.
  • Variablennamen sind case-sensitive (z.B. ist "name" anders als "Name").
  • Vermeiden Sie die Verwendung von Python-Schlüsselwörtern als Variablennamen. Eine Liste der Schlüsselwörter finden Sie in der offiziellen Python-Dokumentation.

Umwandlung von Typen

Manchmal ist es notwendig, den Datentyp einer Variablen zu ändern. Python bietet eingebaute Funktionen für die Typumwandlung:

# Umwandlung von int zu float
x = 5
y = float(x)
print(y)  # Ausgabe: 5.0

# Umwandlung von float zu int
pi = 3.14159
z = int(pi)
print(z)  # Ausgabe: 3

# Umwandlung von int zu str
a = 42
b = str(a)
print(b)  # Ausgabe: "42"

Weitere Informationen zur Typumwandlung finden Sie in der offiziellen Python-Dokumentation.

Type Hints

Python unterstützt Typannotationen (Type Hints), die den erwarteten Typen von Variablen und Funktionsrückgaben angeben. Diese sind besonders nützlich für die Dokumentation und die statische Typprüfung mit Werkzeugen wie mypy.

# Typannotationen für Variablen
age: int = 25
name: str = "Alice"
pi: float = 3.14159

# Typannotationen für Funktionen
def greet(name: str) -> str:
    return f"Hallo, {name}!"

print(greet("Bob"))

Weitere Informationen zu Typannotationen finden Sie in der offiziellen Python-Dokumentation.
