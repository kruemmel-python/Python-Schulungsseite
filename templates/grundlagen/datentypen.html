Python Datentypen

Einführung in Datentypen

Python unterstützt verschiedene grundlegende Datentypen, die zur Speicherung und Manipulation von Daten verwendet werden. Hier sind einige der wichtigsten Datentypen:

Erstellen und Verwenden von Datentypen

Hier sind einige Beispiele für die Erstellung und Verwendung der grundlegenden Datentypen in Python:

# Ganzzahl (Integer)
x = 10
print(type(x))  # Ausgabe: 

# Fließkommazahl (Float)
y = 3.14
print(type(y))  # Ausgabe: 

# Zeichenkette (String)
name = "Alice"
print(type(name))  # Ausgabe: 

# Boolean
flag = True
print(type(flag))  # Ausgabe: 

# Liste
fruits = ["Apfel", "Banane", "Kirsche"]
print(type(fruits))  # Ausgabe: 

# Tupel
coordinates = (10, 20)
print(type(coordinates))  # Ausgabe:

print(type(person))  # Ausgabe:

Weitere Datentypen-Operationen

Python bietet eine Vielzahl von Methoden und Funktionen zur Arbeit mit verschiedenen Datentypen. Hier sind einige nützliche Beispiele:

Arbeiten mit Strings

Strings haben viele nützliche Methoden, wie z.B. upper(), lower(), split(), etc.

# Arbeiten mit Strings
name = "Alice Wonderland"
print(name.upper())  # Ausgabe: ALICE WONDERLAND
print(name.lower())  # Ausgabe: alice wonderland
print(name.split())  # Ausgabe: ['Alice', 'Wonderland']

Arbeiten mit Listen

Listen haben viele nützliche Methoden, wie z.B. append(), remove(), sort(), etc.

# Arbeiten mit Listen
fruits = ["Apfel", "Banane", "Kirsche"]
fruits.append("Orange")
print(fruits)  # Ausgabe: ['Apfel', 'Banane', 'Kirsche', 'Orange']
fruits.remove("Banane")
print(fruits)  # Ausgabe: ['Apfel', 'Kirsche', 'Orange']
fruits.sort()
print(fruits)  # Ausgabe: ['Apfel', 'Kirsche', 'Orange']

Arbeiten mit Dictionaries

Dictionaries haben viele nützliche Methoden, wie z.B. keys(), values(), items(), etc.

# Arbeiten mit Dictionaries
person = {"name": "Alice", "alter": 30}
print(person.keys())  # Ausgabe: dict_keys(['name', 'alter'])
print(person.values())  # Ausgabe: dict_values(['Alice', 30])
print(person.items())  # Ausgabe: dict_items([('name', 'Alice'), ('alter', 30)])
