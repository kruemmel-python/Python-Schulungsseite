Python NumPy

Einführung in NumPy

NumPy (Numerical Python) ist eine Bibliothek für wissenschaftliches Rechnen in Python. Sie bietet Unterstützung für große, mehrdimensionale Arrays und Matrizen, sowie eine Vielzahl von mathematischen Funktionen, um diese effizient zu bearbeiten.

Installation

Um NumPy zu installieren, können Sie den folgenden Befehl verwenden:

pip install numpy

Grundlagen

NumPy Arrays sind die Kernstruktur in NumPy. Sie sind ähnlich wie Listen in Python, aber mit zusätzlichen Funktionen und einer effizienteren Speicherverwaltung.

import numpy as np

# Ein einfaches Array erstellen
array = np.array([1, 2, 3, 4, 5])
print(array)

# Ein 2D-Array erstellen
array_2d = np.array([[1, 2, 3], [4, 5, 6]])
print(array_2d)

Mathematische Operationen

NumPy bietet eine Vielzahl von mathematischen Operationen, die auf Arrays angewendet werden können.

import numpy as np

# Arrays erstellen
a = np.array([1, 2, 3, 4])
b = np.array([5, 6, 7, 8])

# Elementweise Addition
print(a + b)

# Elementweise Multiplikation
print(a * b)

# Matrixmultiplikation
print(np.dot(a.reshape(2, 2), b.reshape(2, 2)))

Erweiterte Funktionen

NumPy bietet auch viele erweiterte Funktionen wie das Erstellen von zufälligen Arrays, Fourier-Transformationen und lineare Algebra.

import numpy as np

# Zufälliges Array erstellen
random_array = np.random.rand(3, 3)
print(random_array)

# Fourier-Transformation
fourier = np.fft.fft(np.array([1, 2, 3, 4]))
print(fourier)

# Lineare Algebra
matrix = np.array([[1, 2], [3, 4]])
inverse = np.linalg.inv(matrix)
print(inverse)

Zusammenfassung

NumPy ist eine leistungsstarke Bibliothek für wissenschaftliches Rechnen in Python. Sie bietet eine Vielzahl von Funktionen und Werkzeugen, die für numerische Berechnungen unerlässlich sind. Weitere Informationen finden Sie in der offiziellen NumPy-Dokumentation.
