Matplotlib

Einführung in Matplotlib

Matplotlib ist eine umfassende Bibliothek zum Erstellen von statischen, animierten und interaktiven Visualisierungen in Python. Weitere Informationen finden Sie in der Matplotlib-Dokumentation.

Beispiele für die Verwendung von Matplotlib

Hier sind einige Beispiele für die Verwendung von Matplotlib in Python:

Einfaches Liniendiagramm

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.xlabel('x-Achse')
plt.ylabel('y-Achse')
plt.title('Einfaches Liniendiagramm')
plt.show()

Balkendiagramm

import matplotlib.pyplot as plt

x = ['A', 'B', 'C', 'D']
y = [10, 20, 25, 30]

plt.bar(x, y)
plt.xlabel('Kategorien')
plt.ylabel('Werte')
plt.title('Einfaches Balkendiagramm')
plt.show()

Streudiagramm

import matplotlib.pyplot as plt

x = [5, 7, 8, 7, 2, 17, 2, 9, 4, 11, 12, 9, 6]
y = [99, 86, 87, 88, 100, 86, 103, 87, 94, 78, 77, 85, 86]

plt.scatter(x, y)
plt.xlabel('x-Werte')
plt.ylabel('y-Werte')
plt.title('Einfaches Streudiagramm')
plt.show()
