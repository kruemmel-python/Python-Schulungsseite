Keras

Einführung in Keras

Keras ist eine benutzerfreundliche API für die Entwicklung und das Training tiefer neuronaler Netze. Weitere Informationen finden Sie in der Keras-Dokumentation.

Beispiele für die Verwendung von Keras

Hier sind einige Beispiele für die Verwendung von Keras in Python:

Ein einfaches neuronales Netzwerk erstellen

from keras.models import Sequential
from keras.layers import Dense

# Modell erstellen
model = Sequential()
model.add(Dense(12, input_dim=8, activation='relu'))
model.add(Dense(8, activation='relu'))
model.add(Dense(1, activation='sigmoid'))

# Modell kompilieren
model.compile(loss='binary_crossentropy', optimizer='adam', metrics=['accuracy'])

# Modell zusammenfassen
model.summary()

Ein Modell trainieren

import numpy as np

# Datengenerierung
X = np.random.random((1000, 8))
y = np.random.randint(2, size=(1000, 1))

# Modell trainieren
model.fit(X, y, epochs=10, batch_size=10)

Ein Modell bewerten

# Modellbewertung
scores = model.evaluate(X, y)
print(f'{model.metrics_names[1]}: {scores[1]*100}')
