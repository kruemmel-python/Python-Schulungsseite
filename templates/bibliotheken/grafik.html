Python Grafik-Bibliotheken

Einführung in Grafik-Bibliotheken

Python bietet verschiedene Bibliotheken für die Erstellung und Bearbeitung von Grafiken. Hier sind einige der beliebtesten und am häufigsten verwendeten Grafik-Bibliotheken in Python. Weitere Informationen finden Sie in der offiziellen Matplotlib-Dokumentation.

Beispiele für die Verwendung von Grafik-Bibliotheken

Hier sind einige Beispiele für die Verwendung von Grafik-Bibliotheken in Python:

Grafiken mit Matplotlib erstellen

Matplotlib ist eine umfangreiche Bibliothek zur Erstellung von statischen, animierten und interaktiven Grafiken in Python.

# Grafiken erstellen mit Matplotlib
import matplotlib.pyplot as plt

# Daten
x = [1, 2, 3, 4, 5]
y = [2, 3, 5, 7, 11]

# Linie zeichnen
plt.plot(x, y)

# Achsenbeschriftungen
plt.xlabel('X-Werte')
plt.ylabel('Y-Werte')

# Titel
plt.title('Einfache Linie')

# Grafik anzeigen
plt.show()

Grafiken mit Seaborn erstellen

Seaborn ist eine Bibliothek zur Erstellung von informativen und attraktiven statistischen Grafiken.

# Grafiken erstellen mit Seaborn
import seaborn as sns
import matplotlib.pyplot as plt

# Daten laden
tips = sns.load_dataset("tips")

# Histogramm zeichnen
sns.histplot(tips, x="total_bill", hue="time", element="step", stat="density")

# Grafik anzeigen
plt.show()

Interaktive Grafiken mit Plotly erstellen

Plotly ist eine Bibliothek zur Erstellung von interaktiven Grafiken und Dashboards.

# Interaktive Grafiken erstellen mit Plotly
import plotly.express as px

# Daten laden
df = px.data.iris()

# Streudiagramm zeichnen
fig = px.scatter(df, x="sepal_width", y="sepal_length", color="species")

# Grafik anzeigen
fig.show()

Zusammenfassung

Python bietet eine Vielzahl von Bibliotheken für die Grafikerstellung, die die Arbeit mit Grafik-Dateien erleichtern. Weitere Informationen finden Sie in der offiziellen Matplotlib-Dokumentation.
