Python Audio-Bibliotheken

Einführung in Audio-Bibliotheken

Python bietet verschiedene Bibliotheken für die Audioverarbeitung. Hier sind einige der beliebtesten und am häufigsten verwendeten Audio-Bibliotheken in Python. Weitere Informationen finden Sie in der offiziellen Python-Dokumentation.

Grundlagen der Audioverarbeitung

Die Audioverarbeitung in Python kann mit verschiedenen Bibliotheken durchgeführt werden, darunter pydub, librosa und wave.

Beispiele für die Verwendung von Audio-Bibliotheken

Hier sind einige Beispiele für die Verwendung von Audio-Bibliotheken in Python:

Audioverarbeitung mit Pydub

Pydub ist eine einfache und benutzerfreundliche Bibliothek zur Bearbeitung von Audio-Dateien.

# Audioverarbeitung mit Pydub
from pydub import AudioSegment

# Audio-Datei laden
audio = AudioSegment.from_file("example.mp3")

# Lautstärke um 6dB erhöhen
louder_audio = audio + 6

# Lautstärke um 3dB verringern
quieter_audio = audio - 3

# Audio-Datei exportieren
louder_audio.export("louder_example.mp3", format="mp3")

Audioanalyse mit Librosa

Librosa ist eine Bibliothek für die Analyse von Musik und Audio.

# Audioanalyse mit Librosa
import librosa

# Audio-Datei laden
y, sr = librosa.load("example.wav")

# Tempo und Beat-Frames extrahieren
tempo, beat_frames = librosa.beat.beat_track(y=y, sr=sr)

print(f"Tempo: {tempo} BPM")

Arbeiten mit dem wave-Modul

Das wave-Modul bietet eine einfache Schnittstelle zum Arbeiten mit WAV-Dateien.

# Arbeiten mit dem wave-Modul
import wave

# WAV-Datei öffnen und lesen
with wave.open("example.wav", "rb") as wav_file:
    # Parameter der WAV-Datei auslesen
    params = wav_file.getparams()
    print(params)

    # Audio-Daten auslesen
    frames = wav_file.readframes(-1)

# Schreiben der Audio-Daten in eine neue WAV-Datei
with wave.open("new_example.wav", "wb") as new_wav_file:
    new_wav_file.setparams(params)
    new_wav_file.writeframes(frames)

Audio mit Soundfile und Numpy laden

Soundfile ist eine Bibliothek zum Lesen und Schreiben von Sound-Dateien, die mit Numpy arbeitet.

# Audio mit Soundfile und Numpy laden
import soundfile as sf
import numpy as np

# Audio-Datei lesen
data, samplerate = sf.read('example.wav')

# Audio-Daten mit Numpy verarbeiten
print(f"Sample Rate: {samplerate}")
print(f"Data Type: {type(data)}")
print(f"Data Shape: {data.shape}")

# Bearbeitete Audio-Daten speichern
sf.write('new_example.wav', data, samplerate)

Spracherkennung mit SpeechRecognition

SpeechRecognition ist eine Bibliothek zur Durchführung von Spracherkennung mit verschiedenen Engines und APIs.

# Spracherkennung mit SpeechRecognition
import speech_recognition as sr

# Initialisieren des Recognizer
recognizer = sr.Recognizer()

# Laden der Audio-Datei
with sr.AudioFile('example.wav') as source:
    audio_data = recognizer.record(source)
    
# Spracherkennung durchführen
try:
    text = recognizer.recognize_google(audio_data)
    print(f"Erkannter Text: {text}")
except sr.UnknownValueError:
    print("Google Speech Recognition konnte den Text nicht verstehen")
except sr.RequestError as e:
    print(f"Konnte keine Verbindung zu Google herstellen; {e}")
