Python Erweiterte Funktionen

Einführung in Erweiterte Funktionen

Python bietet mehrere eingebaute Funktionen, die erweiterte Funktionalitäten ermöglichen, wie map(), filter(), reduce() und lambda-Funktionen. Diese Funktionen sind nützlich für das Verarbeiten und Transformieren von Daten. Weitere Informationen finden Sie in der offiziellen Python-Dokumentation.

Verwendung von map()

Die map()-Funktion wendet eine Funktion auf jedes Element einer iterierbaren Sammlung (z.B. eine Liste) an und gibt eine Karte des Ergebnisses zurück.

zahlen = [1, 2, 3, 4]
quadrate = list(map(lambda x: x ** 2, zahlen))
print(quadrate)  # Ausgabe: [1, 4, 9, 16]

Verwendung von filter()

Die filter()-Funktion filtert Elemente einer iterierbaren Sammlung basierend auf einer Funktion, die entweder True oder False zurückgibt.

zahlen = [1, 2, 3, 4, 5, 6]
gerade_zahlen = list(filter(lambda x: x % 2 == 0, zahlen))
print(gerade_zahlen)  # Ausgabe: [2, 4, 6]

Verwendung von reduce()

Die reduce()-Funktion, die im functools-Modul enthalten ist, wendet eine Funktion kumulativ auf die Elemente einer iterierbaren Sammlung an, sodass sie auf einen einzigen Wert reduziert werden.

from functools import reduce
zahlen = [1, 2, 3, 4]
produkt = reduce(lambda x, y: x * y, zahlen)
print(produkt)  # Ausgabe: 24

Verwendung von lambda-Funktionen

lambda-Funktionen sind kleine anonyme Funktionen, die mit dem Schlüsselwort lambda erstellt werden. Sie können überall dort verwendet werden, wo eine Funktion erforderlich ist.

verdoppeln = lambda x: x * 2
print(verdoppeln(5))  # Ausgabe: 10
