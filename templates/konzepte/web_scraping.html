Python Web Scraping

Einführung in Web Scraping

Web Scraping ist eine Technik, mit der man Daten von Webseiten extrahieren kann. Python bietet mehrere Bibliotheken, um Web Scraping durchzuführen. Weitere Informationen finden Sie in der offiziellen Python-Dokumentation.

Beispiele für Web Scraping

Hier sind einige Beispiele für die Verwendung von Web Scraping in Python:

Web Scraping mit BeautifulSoup

BeautifulSoup ist eine beliebte Bibliothek zum Parsen von HTML und XML in Python.

# Web Scraping mit BeautifulSoup
import requests
from bs4 import BeautifulSoup

# Webseite abrufen
url = 'https://example.com'
response = requests.get(url)

# HTML-Inhalt parsen
soup = BeautifulSoup(response.text, 'html.parser')

# Daten extrahieren
titel = soup.find('title').text
print(titel)

Web Scraping mit Scrapy

Scrapy ist ein leistungsstarkes Web Scraping Framework für Python, das speziell für große Projekte entwickelt wurde.

# Web Scraping mit Scrapy
import scrapy

class QuotesSpider(scrapy.Spider):
    name = "quotes"
    start_urls = [
        'http://quotes.toscrape.com/page/1/',
    ]

    def parse(self, response):
        for quote in response.css('div.quote'):
            yield {
                'text': quote.css('span.text::text').get(),
                'author': quote.css('span small::text').get(),
            }

# Zum Ausführen des Scrapy-Spiders verwenden Sie den Befehl:
# scrapy runspider quotes_spider.py

Web Scraping mit Selenium

Selenium ist ein Web-Testing-Framework, das auch für das Web Scraping verwendet werden kann, besonders wenn JavaScript-Rendering erforderlich ist.

# Web Scraping mit Selenium
from selenium import webdriver
from selenium.webdriver.common.by import By

# Webdriver erstellen
driver = webdriver.Chrome()

# Webseite öffnen
driver.get('https://example.com')

# Daten extrahieren
titel = driver.find_element(By.TAG_NAME, 'title').text
print(titel)

# Webdriver schließen
driver.quit()
