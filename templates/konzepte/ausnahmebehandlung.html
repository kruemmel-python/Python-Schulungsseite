Python Ausnahmebehandlung

Einführung in die Ausnahmebehandlung

Die Ausnahmebehandlung ist ein wichtiges Konzept in der Programmierung, das es ermöglicht, auf Fehler und außergewöhnliche Situationen im Programm zu reagieren. Weitere Informationen zur Ausnahmebehandlung in Python finden Sie in der offiziellen Python-Dokumentation.

Grundlagen der Ausnahmebehandlung

In Python wird die Ausnahmebehandlung mit den Schlüsselwörtern try, except und finally durchgeführt.

Beispiele für Ausnahmebehandlung

Hier sind einige Beispiele für die Verwendung von Ausnahmebehandlung in Python:

Einfacher try-except-Block

Ein einfacher try-except-Block kann verwendet werden, um auf spezifische Fehler zu reagieren.

# Einfacher try-except-Block
try:
    zahl = int(input("Geben Sie eine Zahl ein: "))
    print(f"Die eingegebene Zahl ist {zahl}")
except ValueError:
    print("Das ist keine gültige Zahl!")

Mehrere Ausnahmen behandeln

Sie können mehrere except-Blöcke verwenden, um verschiedene Arten von Ausnahmen zu behandeln.

# Mehrere Ausnahmen behandeln
try:
    zahl = int(input("Geben Sie eine Zahl ein: "))
    ergebnis = 10 / zahl
    print(f"Das Ergebnis ist {ergebnis}")
except ValueError:
    print("Das ist keine gültige Zahl!")
except ZeroDivisionError:
    print("Division durch Null ist nicht erlaubt!")

finally-Block

Der finally-Block wird immer ausgeführt, unabhängig davon, ob eine Ausnahme aufgetreten ist oder nicht.

# finally-Block
try:
    file = open("example.txt", "r")
    content = file.read()
except FileNotFoundError:
    print("Die Datei wurde nicht gefunden!")
finally:
    file.close()

Eigene Ausnahmen definieren

Sie können eigene Ausnahmen definieren, indem Sie eine neue Ausnahmeklasse erstellen, die von der eingebauten Exception-Klasse erbt.

# Eigene Ausnahme definieren
class EigeneAusnahme(Exception):
    pass

def check_value(x):
    if x < 0:
        raise EigeneAusnahme("Der Wert darf nicht negativ sein!")
    return x

try:
    check_value(-1)
except EigeneAusnahme as e:
    print(e)

Zusammenfassung

Die Ausnahmebehandlung in Python ermöglicht es Ihnen, robuste und fehlertolerante Programme zu schreiben. Weitere Informationen zur Ausnahmebehandlung finden Sie in der offiziellen Python-Dokumentation.
