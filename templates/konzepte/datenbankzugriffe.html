Python Datenbankzugriffe

Einführung in Datenbankzugriffe

Datenbankzugriffe sind essenziell für viele Anwendungen, da sie das Speichern und Abrufen von Daten ermöglichen. Python bietet verschiedene Bibliotheken und Frameworks für den Datenbankzugriff. Weitere Informationen finden Sie in der offiziellen Python-Dokumentation.

Beispiele für Datenbankzugriffe

Hier sind einige Beispiele für die Verwendung von Datenbankzugriffen in Python:

Verwendung von SQLite3

SQLite3 ist eine in Python integrierte, leichtgewichtige Datenbank, die ideal für kleinere Anwendungen und Prototypen ist.

# SQLite3 verwenden
import sqlite3

# Verbindung zur Datenbank herstellen
conn = sqlite3.connect('example.db')
c = conn.cursor()

# Tabelle erstellen
c.execute('''CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT, age INTEGER)''')

# Daten einfügen
c.execute("INSERT INTO users (name, age) VALUES ('Alice', 30)")
c.execute("INSERT INTO users (name, age) VALUES ('Bob', 25)")

# Daten abfragen
c.execute("SELECT * FROM users")
print(c.fetchall())

# Änderungen speichern und Verbindung schließen
conn.commit()
conn.close()

Verwendung von SQLAlchemy

SQLAlchemy ist ein umfassendes SQL Toolkit und ORM (Object Relational Mapper) für Python, das die Arbeit mit Datenbanken vereinfacht. 

# Verwendung von SQLAlchemy
from sqlalchemy import create_engine, Column, Integer, String, Sequence
from sqlalchemy.ext.declarative import declarative_base
from sqlalchemy.orm import sessionmaker

# Datenbankverbindung erstellen
engine = create_engine('sqlite:///example.db')
Base = declarative_base()

# Modell erstellen
class User(Base):
    __tablename__ = 'users'
    id = Column(Integer, Sequence('user_id_seq'), primary_key=True)
    name = Column(String(50))
    age = Column(Integer)

# Tabelle erstellen
Base.metadata.create_all(engine)

# Session erstellen
Session = sessionmaker(bind=engine)
session = Session()

# Daten einfügen
new_user = User(name='Charlie', age=35)
session.add(new_user)
session.commit()

# Daten abfragen
for user in session.query(User).all():
    print(user.name, user.age)

# Session schließen
session.close()
