Python Module und Pakete

Einführung in Module und Pakete

Module und Pakete sind essentielle Bestandteile in Python, die es ermöglichen, den Code zu strukturieren und wiederverwendbar zu machen. Weitere Informationen zu Modulen und Paketen in Python finden Sie in der offiziellen Python-Dokumentation.

Grundlagen von Modulen und Paketen

Ein Modul ist eine Datei, die Python-Definitionen und -Anweisungen enthält. Ein Paket ist eine Sammlung von Modulen in einem Verzeichnis.

Beispiele für Module und Pakete

Hier sind einige Beispiele für die Verwendung von Modulen und Paketen in Python:

Einfaches Modul erstellen

Ein Modul kann erstellt werden, indem man eine Python-Datei mit Funktionen und Variablen erstellt.

# Erstellen eines einfachen Moduls (modul.py)
def begruessung(name):
    return f"Hallo, {name}!"

# Verwenden des Moduls in einem anderen Skript
import modul

print(modul.begruessung("Alice"))

Ein Paket erstellen

Ein Paket ist ein Verzeichnis, das eine spezielle __init__.py-Datei enthält, die es Python ermöglicht, das Verzeichnis als Paket zu behandeln.

# Erstellen eines Pakets
# Verzeichnisstruktur:
# meinpaket/
#     __init__.py
#     modul.py

# Inhalt von __init__.py
from .modul import begruessung

# Verwenden des Pakets
import meinpaket

print(meinpaket.begruessung("Bob"))

Standardbibliotheken verwenden

Python bietet viele nützliche Standardbibliotheken, die Sie in Ihren Projekten verwenden können.

# Verwendung der Standardbibliothek
import math

print(math.sqrt(16))

Externe Pakete installieren

Sie können externe Pakete mit dem Paketmanager pip installieren.

# Installation eines externen Pakets
# Öffnen Sie ein Terminal und geben Sie folgendes ein:
# pip install requests

# Verwendung des installierten Pakets
import requests

response = requests.get('https://api.github.com')
print(response.status_code)

Zusammenfassung

Module und Pakete helfen dabei, den Code zu organisieren und wiederverwendbar zu machen. Weitere Informationen zu Modulen und Paketen finden Sie in der offiziellen Python-Dokumentation.
