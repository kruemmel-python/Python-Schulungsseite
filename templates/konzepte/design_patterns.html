Python Design Patterns

Einführung in Design Patterns

Design Patterns sind bewährte Lösungen für häufig auftretende Probleme in der Softwareentwicklung. Sie bieten eine standardisierte Herangehensweise zur Lösung spezifischer Probleme.

Beispiele für Design Patterns

Hier sind einige Beispiele für die Verwendung von Design Patterns in Python:

Singleton Pattern

Das Singleton Pattern stellt sicher, dass eine Klasse nur eine Instanz hat und bietet einen globalen Zugriffspunkt darauf.

# Singleton Pattern
class Singleton:
    _instance = None

    def __new__(cls, *args, **kwargs):
        if not cls._instance:
            cls._instance = super(Singleton, cls).__new__(cls, *args, **kwargs)
        return cls._instance

# Verwendung
s1 = Singleton()
s2 = Singleton()
print(s1 is s2)  # Ausgabe: True

Factory Pattern

Das Factory Pattern bietet eine Möglichkeit zur Erstellung von Objekten, ohne den genauen Klassennamen angeben zu müssen.

# Factory Pattern
class Tier:
    def sprechen(self):
        pass

class Hund(Tier):
    def sprechen(self):
        return "Wau"

class Katze(Tier):
    def sprechen(self):
        return "Miau"

class TierFactory:
    @staticmethod
    def get_tier(tier_typ):
        if tier_typ == "Hund":
            return Hund()
        elif tier_typ == "Katze":
            return Katze()
        return None

# Verwendung
tier = TierFactory.get_tier("Hund")
print(tier.sprechen())  # Ausgabe: Wau

Observer Pattern

Das Observer Pattern definiert eine Abhängigkeit zwischen Objekten, so dass bei einer Zustandsänderung eines Objekts alle abhängigen Objekte benachrichtigt und aktualisiert werden.

# Observer Pattern
class Beobachter:
    def aktualisieren(self, nachricht):
        pass

class Subjekt:
    def __init__(self):
        self._beobachter = []

    def registriere(self, beobachter):
        self._beobachter.append(beobachter)

    def benachrichtige(self, nachricht):
        for beobachter in self._beobachter:
            beobachter.aktualisieren(nachricht)

# Verwendung
class KonkreterBeobachter(Beobachter):
    def aktualisieren(self, nachricht):
        print("Benachrichtigung erhalten:", nachricht)

subjekt = Subjekt()
beobachter1 = KonkreterBeobachter()
beobachter2 = KonkreterBeobachter()

subjekt.registriere(beobachter1)
subjekt.registriere(beobachter2)

subjekt.benachrichtige("Ein Ereignis ist aufgetreten")
