Python Generatoren und Iteratoren

Einführung in Generatoren und Iteratoren

Generatoren und Iteratoren sind leistungsfähige Werkzeuge in Python, die es ermöglichen, Daten sequenziell zu verarbeiten, ohne dass die gesamte Sequenz im Speicher gehalten werden muss. Weitere Informationen zu Generatoren und Iteratoren in Python finden Sie in der offiziellen Python-Dokumentation.

Grundlagen von Generatoren

Generatoren sind eine spezielle Art von Funktionen, die es ermöglichen, Werte nacheinander zu erzeugen, anstatt sie alle auf einmal zurückzugeben. Sie verwenden das Schlüsselwort yield, um einen Wert zurückzugeben und den Zustand der Funktion zu speichern.

Beispiele für Generatoren

Hier sind einige Beispiele für die Verwendung von Generatoren in Python:

Einfacher Generator

Ein Generator, der drei Werte nacheinander erzeugt.

# Einfacher Generator
def einfache_generierung():
    yield 1
    yield 2
    yield 3

for wert in einfache_generierung():
    print(wert)

Generator mit Schleife

Ein Generator, der die Quadratzahlen bis zu einem bestimmten Wert erzeugt.

# Generator mit Schleife
def quadrat_generator(n):
    for i in range(n):
        yield i * i

for wert in quadrat_generator(5):
    print(wert)

Grundlagen von Iteratoren

Ein Iterator ist ein Objekt, das eine Sequenz von Werten nacheinander zurückgeben kann. Ein Iterator implementiert die Methoden __iter__() und __next__().

Beispiele für Iteratoren

Hier sind einige Beispiele für die Verwendung von Iteratoren in Python:

Verwendung von Iteratoren

Ein Iterator, der die Elemente einer Liste nacheinander zurückgibt.

# Verwendung von Iteratoren
meine_liste = [1, 2, 3, 4, 5]
meine_iterator = iter(meine_liste)

while True:
    try:
        element = next(meine_iterator)
        print(element)
    except StopIteration:
        break

Benutzerdefinierter Iterator

Ein benutzerdefinierter Iterator, der eine Sequenz von Werten erzeugt.

# Benutzerdefinierter Iterator
class MeinIterator:
    def __init__(self, max):
        self.max = max
        self.n = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.n < self.max:
            result = self.n
            self.n += 1
            return result
        else:
            raise StopIteration

mein_iterator = MeinIterator(5)
for wert in mein_iterator:
    print(wert)

Zusammenfassung

Generatoren und Iteratoren sind leistungsfähige Werkzeuge, die es ermöglichen, Daten sequenziell und speichereffizient zu verarbeiten. Weitere Informationen zu Generatoren und Iteratoren finden Sie in der offiziellen Python-Dokumentation.
