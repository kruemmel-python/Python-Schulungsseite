Python Dateioperationen

Einführung in Dateioperationen

Dateioperationen sind ein wesentlicher Bestandteil vieler Programme. In Python können Dateien gelesen, geschrieben und manipuliert werden. Weitere Informationen zu Dateioperationen in Python finden Sie in der offiziellen Python-Dokumentation.

Öffnen und Lesen von Dateien

Um eine Datei in Python zu öffnen und ihren Inhalt zu lesen, verwenden Sie die Funktion open(). Der Modus 'r' steht für Lesen.

# Eine Datei öffnen und ihren Inhalt lesen
with open('beispiel.txt', 'r') as file:
    content = file.read()
    print(content)

Schreiben in Dateien

Um in eine Datei zu schreiben, verwenden Sie den Modus 'w'. Wenn die Datei nicht existiert, wird sie erstellt. Wenn sie existiert, wird sie überschrieben.

# Eine Datei im Schreibmodus öffnen und Text schreiben
with open('beispiel.txt', 'w') as file:
    file.write('Hallo, Welt!')

Anhängen an Dateien

Um Text an eine bestehende Datei anzuhängen, verwenden Sie den Modus 'a'.

# Eine Datei im Anhängemodus öffnen und Text anhängen
with open('beispiel.txt', 'a') as file:
    file.write('\nZusätzliche Zeile.')

Zeilenweises Lesen einer Datei

Man kann eine Datei auch zeilenweise lesen, indem man eine Schleife verwendet.

# Zeilenweise Lesen einer Datei
with open('beispiel.txt', 'r') as file:
    for line in file:
        print(line.strip())

Weitere Dateioperationen

Es gibt viele nützliche Funktionen und Methoden für Dateioperationen in Python. Hier sind einige der wichtigsten:

Dateien kopieren

Mit dem Modul shutil kann man Dateien kopieren.

# Eine Datei kopieren
import shutil
shutil.copy('beispiel.txt', 'beispiel_kopie.txt')

Dateien umbenennen

Mit dem Modul os kann man Dateien umbenennen.

# Eine Datei umbenennen
import os
os.rename('beispiel.txt', 'neuer_name.txt')

Dateien löschen

Mit dem Modul os kann man Dateien löschen.

# Eine Datei löschen
import os
os.remove('neuer_name.txt')

Weitere Informationen zu Dateioperationen und den verfügbaren Methoden finden Sie in der offiziellen Python-Dokumentation.
